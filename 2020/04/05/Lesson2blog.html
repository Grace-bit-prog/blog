Github Notes

These notes are what I learned using the Paperspace Platform.

  • Create a Repository in Github

  • In Paperspace, open up a Jupyter terminal

  • Make a new local repository for the files you want to push to your remote Github repository

1) make a test directory where you want to copy your files to.

mkdir test

# cd test

2) initialize the local repository

# git init

Initialized empty Git repository in /notebooks/test/.git/

3) copy the files you need to add to the test directory

4) push your files to your remote github repository

# git push –set-upstream https://github.com/Grace-bit-prog/deeplearning master
